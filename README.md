# 📅 智能排班系统

一个基于 Vue 3 的智能排班管理工具，支持自动排班生成、日历视图、换班调休、Excel 导出等功能。

## ✨ 功能特性

- **排班配置** — 支持两种排班模式：24 小时值班制和分班轮换制，可自定义班次、轮换周期、每日需求人数等参数
- **自动排班** — 根据配置自动生成排班表，支持按组轮换和全员轮转
- **日历视图** — 以月历形式直观展示排班情况，支持按日编辑和请假标记
- **表格视图** — 以表格形式汇总排班数据，支持导出为 Excel 文件
- **调休换班** — 支持两人换班和请假操作，可查看操作历史并撤销
- **数据导入导出** — 支持将全部数据导出为 JSON 文件备份，以及从 JSON 文件恢复
- **本地存储** — 所有数据自动保存在浏览器 localStorage，无需后端服务

## 🛠 技术栈

| 技术 | 说明 |
|------|------|
| [Vue 3](https://vuejs.org/) | 前端框架（Composition API） |
| [Vite](https://vite.dev/) | 构建工具 |
| [Pinia](https://pinia.vuejs.org/) | 状态管理 |

## 🚀 快速开始

### 环境要求

- Node.js >= 18
- npm >= 9

### 安装与运行

```bash
# 克隆项目
git clone https://github.com/aiastia/shift-scheduler.git
cd shift-scheduler

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

### 构建部署

```bash
# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```

## 📖 使用说明

### 1. 排班配置

进入「⚙️ 排班配置」标签页：

1. **添加员工** — 输入姓名和分组（如 A、B、C），点击添加
2. **选择排班模式** — `24h`（全员轮转值班）或 `split`（分组轮班）
3. **设置班次** — 在 split 模式下自定义班次名称和时间
4. **配置参数** — 设置每日需要人数、轮换周期天数、生成天数
5. **选择起始日期** — 排班从哪天开始
6. 点击「生成排班」按钮

### 2. 日历视图

在「📅 排班日历」标签页中：
- 按月查看排班情况，不同员工以颜色区分
- 点击某一天可编辑当日排班或标记请假

### 3. 表格视图

在「📊 表格视图」标签页中：
- 以表格形式查看当月排班汇总
- 显示每人当月值班天数合计
- 支持导出为 `.xlsx` Excel 文件

### 4. 调休换班

在「🔄 调休换班」标签页中：
- **换班** — 选择两位员工和各自的日期进行换班
- **请假** — 选择员工和日期标记请假
- **历史记录** — 查看所有换班/请假操作，支持撤销

### 5. 数据备份

- **导出数据** — 点击顶部「📥 导出数据」按钮，下载 JSON 备份文件
- **导入数据** — 点击顶部「📤 导入数据」按钮，从 JSON 文件恢复数据
